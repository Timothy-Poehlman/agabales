Lightbulb

Motivation:

Lightbulb started as an in class project for an elective CS class that me and my friends had started but with the time for a quarter was not enough to create what we had envisioned for this project. Our main goal was to develop and provide an application that is capable of helping teachers not only in the Computer Science field but a variety of fields. With technology rapidly growing, automating a task that normally teachers do in class would benefit the students and the teachers.

Lightbulb is an online classroom assessment tool that helps instructors assess the performance of their students.

With Lightbulb, instructors are capable of asking different types of questions for students by creating mini quizzes during class.

With the existence of quizzes, instructors are provided classrooms to attach the quizzes they have created. Instructors later on can decide to set a quiz activated.

When setting quizzes active, instructors have the option to require usernames or enable a QR code for students to be able to instantly connect to the quiz during class.

On the student side of things, students are able to enter the classroom that was set active by the instructor. Students just have to answer the questions given to them and submit the quiz at the end.

Lightbulb also allows Mathematical expressions to be displayed using the Latex or Mathjax syntax!

While students are taking the quiz, Lightbulb takes in the data received from students and visualizes the results for the instructor in table and graph format. Although short answer questions have to be corrected by the instructor themselves.

Youtube Playlist to Spotify Playlist Converter

Motivation:

Like other people, we have playlists in Youtube. Unfortunately for me, I had a lot of songs stored in my youtube playlist which I really needed it to be in Spotify as it is more convenient during certain activities i.e. excercising and plus I did not have to Youtube premium so it was not ideal for me. Having 100+ songs, I did not want to manually convert. My laziness in converting it manually led to a fun project haha!

Getting Started:

The game plan I approached was to research the following:
  • Youtube API
  • Spotify API
  • Processing Data between the two APIs

The process

  • Grabbing my API keys from Youtube and Spotify
  • I parsed through the youtube URL and extracted the playlist ID of a given URL. Demo playlist down below!
  • Used the Youtube API to process the ID and retrieve its items
  • Parsed thru the Data given by the Youtube API and put the items in an artist-title format for the Spotify API
  • Fed the artist-title formatted data to the Spotify API to create a new Spotify playlist or add the items into a currently existing playlist

There we go! Songs are now inside a spotify playlist!

Although it is able to convert youtube playlists now, it still needs to get improved as certain youtube titles do not follow the {artist - title} format. In my free time, I make adjustments on parsing to be able to parse more songs that are formatted differently than the rest.

If you want to try out my converter, the github link is in the tab. Feel Free to fork it! I have not posted how to set it up a tutorial on how to use it yet (coming soon) but if you know what you are doing then feel free to try it out.

When a program finishes executing, it displays that program's output. A famous one you already probably know is the terminal in MacOS called Bash.
